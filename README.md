# Simple Ludo Project

A simple collaborative Ad-free Ludo Game.

## Mission
The mission of this project is to create a simple, collaborative, and Ad-free Ludo Game experience for everyone.

## Current Features
* **Offline Multiplayer**: Play with friends and family on the same device.
* **Ad-free Experience**: Enjoy the game without any interruptions.

## Roadmap
* **Computer Play Mode**: Play against an AI opponent.
* **Online Multiplayer**: Connect and play with others over the internet.
